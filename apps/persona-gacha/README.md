# ドコモペルソナガチャ

接客ロープレ用の架空顧客ペルソナを生成する研修支援ツールです。

## 概要

本ツールは、実在顧客情報を扱わず、架空の顧客ペルソナをブラウザ上で生成します。

AI APIは使用しないため、利用回数に応じた従量課金は発生しません。

生成されたペルソナをもとに、接客時のヒアリング力・確認力・社内相談力を高めることを目的としています。

## 主な機能

- タイプ別ランダム生成
- おまかせガチャ
- 難易度指定ガチャ
- 講師ロープレモード
- ひとり学習モード
- 生徒用QR共有
- ロープレタイマー
- 基本的な破綻チェック

## モード

### 講師ロープレモード

講師画面には、基本情報とヒアリング結果が表示されます。

生徒にはQRで `student.html` を共有し、基本情報のみ表示します。

### ひとり学習モード

基本情報のみ表示した状態から始まり、必要なヒアリング項目を段階的に開示できます。

## v1の方針

- ログインなし
- DBなし
- AI APIなし
- PDF保存なし
- GitHub Pagesで公開想定
