# ShiftCore Suite

ShiftCore関連アプリをまとめる統合リポジトリです。

## 統合対象

- Account Console
- PMO / Pick My Off
- OrderCase
- Persona Gacha

## 統合対象外

- Hirotimer

Hirotimerは単体ツールとして独立管理します。

## 現在の移行方針

まずは既存リポジトリを削除せず、ファイルコピーで安全に統合します。

Git履歴の統合は行いません。
